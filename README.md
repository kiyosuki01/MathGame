# Math Game

Console-based math game written in C#.

## Features
- Play addition, subtraction, multiplication, division.
- Random operation mode.
- Difficulty levels: Easy, Normal, Hard.
- Track game history.
- Time tracking per game.

## How to run
**Requirements:** .NET 6.0 SDK or higher installed
1. Clone the repository:
    ```bash
    git clone https://github.com/kiyosuki01/MathGame
    ```
2. Open the terminal in project folder.
3. Run the game:
    ```bash
    dotnet run
    ```
4. Follow the on-screen instructions to play.

## Controls
- Choose operation from main menu
- Input answers using keyboard
- Navigate menu using number keys
